# Project Overview

## Introduction

CyberJay Holdings is a fictional enterprise technology company designed to demonstrate the implementation of a secure, scalable, and highly available Enterprise Wide Area Network (WAN). The project simulates how a modern organization connects multiple geographical locations while maintaining centralized management, security, and reliable communication.

The network is developed using Cisco Packet Tracer and follows Cisco Enterprise Network Design principles. It incorporates hierarchical network architecture, VLAN segmentation, VLSM, OSPF routing, redundancy, and centralized server services using Windows Server 2019.

---

# Company Background

CyberJay Holdings specializes in providing enterprise technology solutions to businesses across different regions. The company offers professional services in networking, cybersecurity, software development, hardware supply, and managed IT services.

To efficiently support customers and internal operations, CyberJay Holdings operates from one headquarters and three strategic branches.

# Project Goals

The project aims to:

* Design a scalable enterprise WAN.
* Connect all company branches securely.
* Centralize enterprise services at Headquarters.
* Implement VLAN segmentation.
* Apply Variable Length Subnet Masking (VLSM).
* Configure OSPF dynamic routing.
* Improve redundancy and fault tolerance.
* Secure communication using enterprise networking best practices.
* Demonstrate practical networking skills suitable for academic assessment and professional portfolio development.

---

# Expected Outcome

Upon completion, the project will simulate a fully operational enterprise WAN capable of supporting secure communication between all CyberJay Holdings locations while providing centralized authentication, routing, file services, and network management.
